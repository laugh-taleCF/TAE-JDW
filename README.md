# MATLAB音频水印项目

## 项目描述
这是一个基于MATLAB的音频数字水印系统，实现音频信息的加密和水印嵌入。

## 主要功能
- 音频水印生成与嵌入
- 音频加密/解密
- 水印提取与验证

## 文件说明
- `main.m` - 主程序入口
- `S_Gen_Audio.m` - 音频水印生成
- `Sk_Gen.m` - 密钥生成
- `.gitignore` - 忽略临时文件

## 使用方法
1. 运行 `main.m`
2. 按照提示操作

## 依赖
- MATLAB R2020a或更高版本
- 信号处理工具箱
